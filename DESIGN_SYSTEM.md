# IGNIVANCE - WEB DEVELOPMENT AGENCY DESIGN SYSTEM

## COLOR PALETTE

### Primary Colors
- **White**: #FFFFFF (Primary background)
- **Slate 900**: #0F172A (Primary text, brand)
- **Blue 500**: #3B82F6 (Accent, CTAs)

### Neutral Grays
- **Gray 50**: #F8FAFC (Subtle backgrounds)
- **Gray 100**: #F1F5F9 (Card backgrounds)
- **Gray 200**: #E2E8F0 (Borders)
- **Gray 600**: #475569 (Secondary text)

### Usage Rules
- White background everywhere
- Blue only for CTAs and accents
- Gray for borders (never shadows)
- High contrast for readability

---

## TYPOGRAPHY

### Font Family
- **Primary**: System font stack (SF Pro, Segoe UI, Roboto)
- **Fallback**: -apple-system, sans-serif

### Scale
- **Hero**: 3.75rem (60px) - Bold
- **H2**: 2.25rem (36px) - Bold
- **H3**: 1.5rem (24px) - Bold
- **Body Large**: 1.25rem (20px) - Regular
- **Body**: 1rem (16px) - Regular

### Line Height
- Headings: 1.2
- Body: 1.6

---

## SPACING SYSTEM

### Section Padding
- Desktop: py-24 (96px)
- Mobile: py-16 (64px)

### Container
- Max width: 1152px (max-w-6xl)
- Padding: px-6 (24px)

### Component Spacing
- Between sections: 96px
- Card padding: 32px
- Button padding: 16px 32px

---

## BORDER RADIUS

### Scale
- **Small**: 0.5rem (8px) - Badges
- **Medium**: 0.75rem (12px) - Buttons
- **Large**: 1rem (16px) - Cards
- **XL**: 1.5rem (24px) - Hero cards

### Usage
- Consistent across all components
- Never mix different radii

---

**Secondary**
- Background: Gray 100

---

## LAYOUT PRINCIPLES

### Grid System
- 2 columns on mobile
- 3-4 columns on desktop
- Gap: 24px (gap-6)

### Whitespace
- Generous padding
- Clear visual hierarchy
- Breathing room between elements

### Alignment
- Left-aligned text for readability
- Center-aligned headings
- Consistent margins

---

## PERFORMANCE RULES

### NEVER USE
❌ Scroll animations
❌ Intersection Observer
❌ 3D effects
❌ Parallax
❌ Typing animations
❌ Heavy shadows
❌ Backdrop blur
❌ Gradient text

### ALWAYS USE
✅ Static layouts
✅ Solid backgrounds
✅ Simple borders
✅ Instant rendering
✅ Smooth scrolling

---
